# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release
- Menu bar display with sun icon and usage percentages (5h/7d)
- Detailed popover with progress bars for all usage limits
- Session (5h), Weekly (7d), and Sonnet Weekly tracking
- Color-coded indicators (green < 50%, yellow 50-80%, red > 80%)
- Time until reset display
- Auto-refresh every minute
- Manual refresh via button or right-click menu
- Right-click context menu with Refresh and Quit options
- Custom sun app icon (SF Symbol based)
